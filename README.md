# Student Placement Prediction (ML Project)
. Overview
This project predicts whether a student will get placed or not based on:
- CGPA  
- IQ  
- Communication Skills  
- Number of Projects  
- Internship Experience  
The model is built using Logistic Regression.
---
## Tech Stack
- Python  
- Pandas  
- Scikit-learn  
- Google Colab  
---
## Model Used
Logistic Regression
---
## Output
The system predicts:
- Placed  
-  Not Placed  
---
## How to Run
1. Open the notebook in Google Colab  
2. Run all cells  
3. Enter student details  
4. View placement prediction
